## Manifest Sync Rule

After every manifest write (updating `publishing-house/manifest.yaml`), immediately sync to the portal:

   > "Portal sync failed. Your local manifest is safe. The portal will catch up on next refresh."

- After intake completes and manifest is first written

- If MCP is unavailable (local-only mode per MCP Availability Check)
